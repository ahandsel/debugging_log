# DaVinci Resolve Tips & Notes

## DaVinci Shortcuts

"K" for forward
"J" for rewind
Double click "M" for Markers

Jump from Marker to Marker: Shift + UP Arrow

## Export DaVinci Resolve Projects

Project > `Export Project Archive` > Included Media --> Full Backup of the project
