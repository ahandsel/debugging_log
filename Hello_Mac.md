## Packages to Install

### Homebrew - Package Manager
* [Homebrew](https://brew.sh/) or `brew` is a package manager for MacOS.

```shell
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

### Packages

| Package                                                         | brew                                                                        | Note                                    |
| --------------------------------------------------------------- | --------------------------------------------------------------------------- | --------------------------------------- |
