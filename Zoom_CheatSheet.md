# Zoom Shortcuts (Custom)

| Shortcut | Action                              |
| -------- | ----------------------------------- |
| `Space`  | Temp Unmute                         |
| `⇧⌘E`    | Pause / Resume Screen Sharing       |
| `⇧⌘W`    | Mute/Unmute My Audio                |
| `⇧⌘D`    | Show/Hide In-meeting Chat Panel     |
| `⇧⌘X`    | Raise/Lower Hand                    |
| `⇧⌘F`    | Show/Hide Floating Meeting Controls |
